Welcome to
RestMan Installer

This wizard will set up your app, install all dependencies,
configure it to run 24/7, and enable automatic updates.

What this installer does:
✦ Clones the RestMan repo & installs services if needed
✦ Installs frontend & backend dependencies
✦ Configures your environment variables locally
✦ Ensures the app is running automatically on reboots
✦ Sets up auto-updates from GitHub every 30 minutes
https://github.com/nithin-sivakumar/open-restman

Your app is live!
PM2 is running your app 24/7.
It will start automatically on reboot and auto-update from GitHub.
Useful PM2 commands:
pm2 list — view running processes
pm2 logs restman-app — stream backend logs
pm2 restart all — restart everything
pm2 monit — live monitoring dashboard
pm2 delete all — stop restman service
